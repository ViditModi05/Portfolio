ℹ️ Description

Antpocalypse is a pixel-art tower defense game where you command a colony of black ants defending their queen against waves of invading insects. From rival red ants to armored beetles, each enemy brings unique challenges. Players must strategically place and upgrade ant units, manage resources, and survive increasing difficulty levels in this quirky but intense survival experience.

✅ Contributions

  • Designed and coded the core tower defense mechanics.
  • Created enemy wave system with increasing difficulty.
  • Managed UI elements like health bars, resource counters, and wave timers.
  • Developed and balanced gameplay across multiple levels
