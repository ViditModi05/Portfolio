ℹ️ Description

Hexa Sort is a color-based puzzle game where players must sort hexagonal tiles into their correct containers. With increasing complexity, smooth animations, and vibrant visuals, it challenges both logic and patience. The gameplay is designed to be casual yet rewarding, offering a perfect balance between fun and mental stimulation.

✅ Contributions

  • Designed core sorting mechanics and level progression.
  • Created intuitive drag-and-drop system
  • Implemented animation and color feedback for player actions.
